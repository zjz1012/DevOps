# DevOps  
为收集和修改的日常用sh脚本  
目录如下  
1、Denial of Service.sh  
DOS攻击行为防火墙IP屏蔽  
2、PID.sh  
根据用户输入的PID，过滤出该PID所有的信息  
3、bkDatabaseName.sh  
Mysql数据库备份和压缩  
4、list_sys_status.sh  
显示主机名、IP地址、子网掩码、网关、DNS服务器IP地址信息  
5、nmap.sh	Create nmap.sh  
扫描网段内主机存活，nmap扫描  
6、ordatabak.sh  
ORACLE数据库备份和压缩  
7、pc_noline.sh  
另一种扫描网段内主机存活  
8、slave_status.sh  
mysql数据库情况反馈  
9、user.sh  
根据用户名查询该用户的所有信息  
10、very.ser.sh  
检查服务vsftpd httpd sshd crond、mysql中任意一个服务的状态
